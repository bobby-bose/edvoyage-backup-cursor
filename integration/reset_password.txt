Reset Password Integration Plan
==============================

Flutter Page: lib/screens/login/reset_password.dart

1. API Endpoint:
   - Use endpoint: /api/auth/reset_password/ (or as defined in backend)
   - Add endpoint constant in lib/constants/api_endpoints.dart

2. API Call:
   - Use http package for POST request to reset password endpoint
   - Send email/username as JSON
   - Handle response codes (200: success, 400/404: error)
   - Show confirmation message on success

3. State Management:
   - Use provider/riverpod for auth state if needed
   - Show loading indicator during request
   - Show error message on failure (SnackBar/AlertDialog)

4. Debugging:
   - Print debug info in kDebugMode (method, status, response)

5. Null Safety:
   - Check for nulls in response and user input

6. UI:
   - Disable reset button while loading
   - Show success message and navigate to login on success

7. Error Handling:
   - Handle SocketException, TimeoutException, FormatException
   - Show user-friendly error messages

8. Testing:
   - Test with Postman and Django backend

Refer to integration_guidelines.txt for standards.
